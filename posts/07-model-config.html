模型配置指南：从GPT到国产大模型的全面支持

飘雪思考 · OpenClaw从入门到精通系列

模型决定了你的虾有多聪明

OpenClaw本身不包含AI模型——它是一个"Agent系统"，需要外接大模型提供智能。选哪个模型、怎么配置、花多少钱，直接决定了你的Agent有多强。

好消息是：OpenClaw支持几乎所有主流模型，从$25/百万token的Claude Opus到完全免费的本地Ollama，丰俭由人。

三种配置方式：
- 内置Provider：Anthropic、OpenAI、Google、智谱——只需设置API Key
- 自定义Provider：DeepSeek、豆包、Kimi等——需要手动添加配置
- 本地模型：Ollama、LM Studio——OpenClaw自动发现

国产模型：性价比之王

DeepSeek：价格只有Claude的1/20

输入价$0.14/百万token，是Claude Sonnet的二十分之一。2026年3月发布的V4是万亿参数级旗舰，支持1M上下文。

注意： DeepSeek在高峰期偶尔延迟或不可用，不建议作为唯一Provider，必须搭配Fallback模型。

智谱GLM：内置支持，配置最简

OpenClaw内置了zai Provider，一条命令完成配置：

openclaw onboard --auth-choice zai-api-key

GLM-5是开源模型中SWE-bench得分最高的，输入价仅$0.80。GLM Flash系列完全免费，适合心跳任务和简单对话。

通义千问：每天2000次免费

通过插件+OAuth接入，不需要API Key：

openclaw plugins enable qwen-portal-auth

Qwen 3.5 Coder专为代码场景设计，$0.22/百万token，极致性价比。

五种成本方案

方案1：极致省钱（月均<$5）

  • 主力：DeepSeek-V3.2（$0.14/$0.28）
  • 备选：Qwen 3.5 Plus（$0.40/$1.20）
  • 心跳任务：GLM-4.5-Flash（免费）

方案2：国产均衡（月均$5-15）

  • 主力：GLM-5（$0.80/$2.56）
  • 备选：DeepSeek-V3.2（$0.14/$0.28）
  • 推理任务：Kimi K2.5（$0.60/$3.00）

方案3：国际均衡（月均$10-30）

  • 主力：Claude Sonnet 4.6（$3.00/$15.00）
  • 轻量：Claude Haiku 4.5或Gemini Flash
  • 复杂任务：按需升级Claude Opus 4.6

方案4：混合最优（月均$5-20，推荐）

  • 复杂任务：Claude Sonnet 4.6
  • 日常对话：DeepSeek-V3.2
  • 心跳定时：Gemini Flash或本地Ollama
  • Fallback链：Sonnet -> Haiku -> DeepSeek-V3.2

方案5：完全免费

  • 方案A：本地Ollama + Qwen3.5-Coder:32B（需32GB内存）
  • 方案B：免费API组合——GLM-4.5-Flash + ERNIE Speed + Gemini Flash

Fallback链：最重要的省钱策略

Fallback链让OpenClaw在主模型不可用时自动切换到备用模型。更重要的是，它可以按任务复杂度路由到不同价位的模型。

策略 模型 相对成本
全部Claude Sonnet 单一模型 100%（基线）
Sonnet + Haiku 二级Fallback 约50-60%
Sonnet -> Haiku -> DeepSeek 三级Fallback 约5-20%
纯DeepSeek 单一低价 约5%

从Claude Sonnet切换到三级Fallback链，可以降低80-95%的API成本。 大部分简单任务（问候、天气、简单查询）自动路由到最便宜的模型。

预算上限：必须设置

{
  "agents": {
    "defaults": {
      "budget": {
        "maxTokensPerDay": 500000,
        "maxCostPerDay": 5.00
      }
    }
  }
}

强烈建议所有用户设置每日预算上限。 即使不差钱，$5/天的限制也能在Agent进入推理循环时保护你的钱包。
